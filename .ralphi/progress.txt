## Codebase Patterns
- **Project detection**: `detectProject(pi, cwd)` in `src/project.ts`; uses `pi.exec('git', ['remote', 'get-url', 'origin'], { cwd })` - passes `cwd` option so git runs in the correct directory; SHA256 hash via `node:crypto` `createHash('sha256').update(input).digest('hex').substring(0, 12)`; `GLOBAL_PROJECT_ID = 'global'` constant for non-git fallback
- **pi.exec mocking in tests**: create minimal mock `{ exec: vi.fn(...) } as unknown as ExtensionAPI`; chain `.mockResolvedValueOnce()` for sequential call behaviors; ExecResult shape: `{ code, stdout, stderr, killed }`
- **Config module**: `loadConfig()` in `src/config.ts`; imports `Config` type from `./types.js`; uses TypeBox `Type.Partial(Type.Object(...))` + `Value.Clean()` for runtime validation of partial overrides; `DEFAULT_CONFIG` and `CONFIG_PATH` are named exports
- **ESLint Node globals**: `eslint.config.js` imports `globals` from the `globals` package and spreads `globals.node` into `languageOptions.globals` to allow `console`, `Buffer`, `process`, etc.
- **fs mock pattern**: In vitest, `vi.mock("node:fs")` + `vi.mocked(fs).existsSync.mockReturnValue(...)` + cast mock return as `unknown as ReturnType<typeof fs.readFileSync>` to satisfy TypeScript

- **Pi Extension structure**: Default export `function(_pi: ExtensionAPI): void` in `src/index.ts`; pi-package requires `keywords: ["pi-package"]` and `pi.extensions` in package.json
- **Module resolution**: `"module": "NodeNext"` + `"moduleResolution": "NodeNext"` in tsconfig; use `.js` extension in relative imports (e.g., `import from "./index.js"`)
- **Peer deps available**: All `@mariozechner/*` packages (pi-coding-agent, pi-ai, pi-tui) and `@sinclair/typebox` are installed as peer deps via npm auto-install
- **ESLint flat config**: `eslint.config.js` imports `@eslint/js` and `@typescript-eslint/eslint-plugin` directly; `files: ["src/**/*.ts"]` pattern scopes TS rules
- **Run checks directly**: `node_modules/.bin/vitest run`, `node_modules/.bin/eslint src/`, `node_modules/.bin/tsc --noEmit` for raw output (pi output filter wraps npm scripts)
