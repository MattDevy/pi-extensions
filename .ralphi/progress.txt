## Codebase Patterns
- **Pi Extension structure**: Default export `function(_pi: ExtensionAPI): void` in `src/index.ts`; pi-package requires `keywords: ["pi-package"]` and `pi.extensions` in package.json
- **Module resolution**: `"module": "NodeNext"` + `"moduleResolution": "NodeNext"` in tsconfig; use `.js` extension in relative imports (e.g., `import from "./index.js"`)
- **Peer deps available**: All `@mariozechner/*` packages (pi-coding-agent, pi-ai, pi-tui) and `@sinclair/typebox` are installed as peer deps via npm auto-install
- **ESLint flat config**: `eslint.config.js` imports `@eslint/js` and `@typescript-eslint/eslint-plugin` directly; `files: ["src/**/*.ts"]` pattern scopes TS rules
- **Run checks directly**: `node_modules/.bin/vitest run`, `node_modules/.bin/eslint src/`, `node_modules/.bin/tsc --noEmit` for raw output (pi output filter wraps npm scripts)
